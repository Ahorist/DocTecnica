Instalación

Instalar VSCode

Para instalar, sigue estos pasos:

  1. Ve al sitio web para descargar VSCode
  2. Haz clic en el bóton "Descargar" para la versión correspondiente a tu sistema operativo (Windows, macOS o Linux)
  3. Ejecuta el instalador y sigue las instrucciones para instalar VSCode

Instalar Node.js

  1. Ve al sitio web de Node.js
  2. Haz clic en el botón "Descargar" para la versión correspondiente a tu sistema operativo (Windows, macOS o Linux)
  3. Ejecuta el instalador y sigue las instrucciones para instalar Node.js
  4. Corroborar que Node.js este bien instalado de la siguiente forma: • Abre una terminal o línea de comandos en tu sistema operativo. • escribe el siguiente comando para verificar la versión de Node.js instalada:
node -v

• Presiona Enter. Deberías ver la versión de Node.js instalada, por ejemplo:

v14.17.0

Crea un Nuevo proyecto en VSCode

  1. Abre VSCode

  2. Haz clic en "Archivo">"Nuevo archivo" o presiona Ctrl + N (Windows/Linux) o Cmd + N (macOS)
  3. Guarda el archivo con el nombre "hola_mundo.js"

Escribir el Código "Hola Mundo" en JavaScript

  1. En el archivo "hola_mundo.js" escribe el siguiente código:
    2. console.log("Hola Mundo");
  2. Guarda el archivo

Ejecutar el Código

  1. Abre una terminar en VSCode presionando Ctrl + ñ
  2. Escriba el siguiente comando para ejecutar el código:
    3. node hola_mundo.js
  3. Presiona Enter para ejecutar el comando
  4. Felicidades ya podrias visualizar tu resultado. Logrando con exito tu primer "Hola Mundo" en JS y de esta forma comprobando que la instalación fue todo un exito.
Uso Básico

Variables

• Una variable es un contenedor que almacena un valor.

• Se declara una variable utilizando la palabra clave let o const.

• Ejemplo:let nombre = 'Juan';

Tipos de Datos

• Números: Representan valores numéricos, como 1 2 3, etc.

• Cadenas: Representan texto, como 'Hola Mundo'

• Booleanos: Representan valores verdaderos o falsos, como true o false.

• Nulos: Representan la ausencia de un valor, como null.

• Indefinidos: Representan un valor que no ha sido asignado, como undefined.

Operadores

• Asignación: =, asigna un valor a una variable.

• Suma: +, suma dos valores.

• Resta: -, resta un valor de otro.

• Multiplicación: *, multiplica dos valores.

• División: /, divide un valor entre otro.

    Ejemplos:

  • let edad = 25;
  • let resultado = 2 + 3;
  • let saludo = 'Hola ' + 'Mundo';
Funciones Avanzadas

Callbacks

¿Que son las Callbacks?

Una callback es una función que se pasa como argumento a otro función, y se ejecuta cuando se completa una operación o se produce un evento específico. En otras palabras, una callback es una función que se llama "de vuelta" (de ahi el nombre) cuando se produce un evento o se completa una operación.

¿Cómo funcionan las callbacks?

Aquí hay una explicación paso a paso de cómo funcionan las callbacks:

  1. Se define una función principal que realiza una operación o llama a otra función.
  2. Se define una función de callback que se pasará como argumento a la función principal.
  3. La función principal recibe la función de callback como argumento y la amacena en una variable.
  4. Cuando se completa la operación o se produce el evento específico, la función principal llama a la función de callback.
  5. La función de callback se ejecuta con los argumentos que se le pasan desde la función principal.

Características clave de las callbacks

  • Las callbacks se ejecutan de forma asíncrona, es decir, no bloquean el flujo de ejecucón del programa.
  • Las callbacks pueden recibir argumentos desde la función principal.
  • Las callbacks pueden retornar valores a la función principal.

Ventajas de las callbacks

  • Permiten manejar operaciones asíncronas de manera efectiva
  • Permiten separar la lógica de la función principal de la lógica de la función de callback.
  • Son útiles para manejar errores y excepciones.

Ejemplo de Callback

function suma(a, b, callback){ let resultado = a + b; callback(resultado); } suma(2, 3, function(resultado){ console.log(resultado); //imprime 5 });

En este ejemplo, la función suma toma tres argumentos:a, b y callback. La función suma realiza la operación de suma y luego llama a la función callback pasando el resultado como argumento.

La función callback es una función anónima que se define en el momento de llamar a la función suma. Esta función toma un argumento resultado y lo imprime en la consola.
